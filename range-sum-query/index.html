🌳 Range Sum Query — Mutable

Efficiently update elements and compute range sums using a Binary Indexed Tree (Fenwick Tree). Click cells to update, drag to select ranges!

Problem Statement
Given an integer array nums, handle two types of queries:
  • update(index, val) — set nums[index] to val
  • sumRange(left, right) — return the sum of elements from index left to right (inclusive)
Key Insight: A naive approach gives O(n) for either update or query. A Binary Indexed Tree (BIT / Fenwick Tree) achieves O(log n) for both operations!

Data Structure Visualization
Structure
A binary tree where each node stores the sum of its range. Leaves hold individual elements.
Update
Change a leaf, then propagate upward — each parent recalculates as the sum of its children.
Query
Traverse the tree, combining nodes whose ranges are fully within the query range.
Key Trick
Use i & (-i) to find the lowest set bit. This determines which elements each tree node is responsible for.
Update Path
Add i += i & (-i) to walk up the tree, updating all ancestor nodes.
Query Path
Subtract i -= i & (-i) to accumulate prefix sums from ancestors.
